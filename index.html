Password Keypad Door Lock

A smart security system that unlocks a door using a keypad password and Arduino controller.

Overview

The Smart Keypad Door Lock is an embedded system project designed to improve home security. Users must enter a correct password using a keypad to unlock the door. If the password is correct, the servo motor rotates and unlocks the door.

If the password is incorrect, the system denies access and displays a warning. After several incorrect attempts, the system can trigger an alarm.

Hardware Components

  • Arduino Uno
  • 4x4 Matrix Keypad
  • Servo Motor
  • 16x2 LCD Display
  • LED Indicators
  • Breadboard
  • Jumper Wires

Software Logic

The Arduino program reads keypad input and compares it with the stored password. If the password matches, the servo motor rotates to unlock the door. 


if(password == enteredPassword){
servo.write(90);
lcd.print("Access Granted");
}

else{
lcd.print("Access Denied");
}
